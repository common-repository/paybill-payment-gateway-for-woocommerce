PayBill.NG Woocommerce Payment Gateway adds PayBill.NG checkout option

== Description ==
For Nigeria Only,PayBill.NG Woocommerce Payment Gateway gives you the option to checkout with PayBill.NG and pay with our payment gateway.

== Installation ==

1. Upload `paybill-payment-gateway-plugin-for-woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on settings option that shows up once the plugin is activated, this will redirect you to woocommerce checkout with PayBill.NG Payment Gateway tab.
4. Update PayBill.NG Payment Gateway form with the information gotten from http://paybill.ng settings page.
5. Done!
